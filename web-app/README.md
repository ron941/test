# Productivity Planner

Productivity Planner 是一款以純 Node.js 打造的輕量級任務規劃工具，透過清晰的 UI 與即時互動協助你在每日工作中保持節奏。此專案包含使用 Node 原生模組撰寫的後端 API 與原生 JavaScript 前端程式碼，可直接於本機啟動並體驗完整功能。

## 環境需求

- Node.js 18 或更新版本（內建 npm 即可）

## 安裝與執行

1. 進入專案資料夾：

   ```bash
   cd web-app
   ```

2. 本專案未使用額外 npm 套件，因此無需安裝依賴。

3. 啟動開發伺服器（預設為 http://localhost:3000 ）：

   ```bash
   npm start
   ```

4. 開啟瀏覽器造訪 `http://localhost:3000` 即可使用 Productivity Planner。

## 功能導覽

- **新增任務**：在「新增任務」區塊輸入標題與描述後按下「加入任務」，任務即會儲存在伺服器端記憶體陣列中。
- **更新完成狀態**：勾選任務左側的核取方塊即可切換完成狀態。
- **刪除任務**：點選任務卡片上的「刪除」按鈕或長按開啟選單，可移除不需要的任務。
- **專注洞察**：右側顯示完成率與進度摘要，協助快速掌握今日節奏。
- **重新整理**：點擊「重新整理」按鈕可手動載入最新任務資料。

> 注意：此示範應用僅使用記憶體儲存資料，伺服器重啟後資料會重置。

## 專案結構

```
web-app/

│   ├── styles.css      # 介面樣式
│   └── app.js          # 前端互動邏輯
└── README.md           # 使用說明
```

## 開發筆記

- 採用 Node.js 原生模組（`http`、`fs`、`url` 等）建構 RESTful API 與靜態檔案伺服器，避免對外部套件依賴。
- 前端使用原生 JavaScript 及模板元素，維持簡潔的互動邏輯。
- 若要整合資料庫或持久化儲存，可在 `server.js` 中替換記憶體陣列實作。
